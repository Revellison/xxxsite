# История задач и правок

Этот файл фиксирует, какие задачи были поставлены в рамках разработки сайта и какие изменения были выполнены по шагам.

## Этап 1. Первичный запуск проекта (многостраничный сайт-пособие)

### Что было запрошено

- Создать сайт на тему: тематическое пособие по информатике (школьный курс).
- Реализовать многостраничную структуру с React Router.
- Сделать боковую панель с группировкой тем по классам.
- На каждую тему сделать отдельную страницу с минимальным наполнением.
- Добавить плейсхолдеры изображений https://placehold.co/600x400 и подписи, какие изображения нужны.
- Сделать дизайн в черно-белом стиле и добавить плавные анимации.
- Использовать стек: React, JavaScript, CSS modules, React Router.
- Создать metod.md для последующего углубления тем через ИИ.

### Что было сделано

- Собрана структура данных тем по блокам:
  - 1-4 классы
  - 5-6 классы
  - 7-9 классы
  - 10-11 классы
- Реализованы страницы:
  - Главная со списком тем и фильтрацией по классам
  - Отдельная страница темы
- Реализована боковая навигация с активным состоянием темы.
- Добавлены плавные анимации в интерфейсе.
- Добавлен методический файл metod.md с шаблонами промптов.
- Проверена сборка проекта: успешно.

## Этап 2. Первый пакет правок по UI и контенту

### Что было запрошено

1. Убрать градиенты, оставить только черно-белую гамму (допуская темно-серые акценты).
2. На страницах тем сделать вид как у markdown-документа и лучше рендерить текст из md-файлов.
3. Полностью переключить шрифт на Rubik.
4. Для мобильной версии сделать Sidebar через burger.

### Что было сделано

- Убраны градиенты из основных блоков интерфейса и фона.
- Подключен шрифт Rubik и применен глобально.
- Страница темы переведена на рендер markdown через react-markdown.
- Созданы отдельные markdown-файлы для тем в каталоге src/content/topics.
- Реализован мобильный drawer Sidebar с burger-кнопкой, overlay и закрытием по клику.
- Проверена сборка проекта: успешно.
